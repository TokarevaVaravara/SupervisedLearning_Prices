# 🏠 Rental Price Prediction — Linear Regression from Scratch

Исследовательский ML-проект по предсказанию цены аренды квартир в Нью-Йорке. Реализованы линейная регрессия и методы регуляризации с нуля на NumPy, с полным сравнением против sklearn-реализаций.

---

## Задача

Датасет: листинги аренды недвижимости в Нью-Йорке (RentHop, 2016).  
**Целевая переменная:** цена аренды (`price`).  
**Тип задачи:** регрессия.

---

## Стек

- Scikit-learn (для сравнения с эталонными реализациями)

## Что сделано

### 1. Анализ данных и подготовка признаков
- Удаление выбросов по цене (1-й и 99-й перцентили)
- Извлечение топ-20 признаков из текстового поля `features` (Elevator, Doorman, Hardwood Floors и др.) с кодированием в бинарные флаги
- Итоговый набор: 22 признака (20 бинарных + bathrooms + bedrooms)
- Разбивка train/test: 80/20, random_state=42

### 2. Реализация моделей с нуля

**Линейная регрессия:**
- Аналитическое решение (нормальное уравнение): `w = (X^T X)^{-1} X^T y`
- Градиентный спуск (Batch GD)
- Стохастический градиентный спуск (SGD)
- Mini-batch градиентный спуск

**Регуляризованные модели (SGD):**
- Ridge (L2): штраф `α * ||w||²`
- Lasso (L1): штраф `α * |w|`
- ElasticNet (L1 + L2): комбинация с параметром `l1_ratio`

**Скейлеры:**
- MinMaxScaler (своя реализация + сравнение со sklearn)
- StandardScaler (своя реализация + сравнение со sklearn)

### 3. Эксперименты

- Сравнение всех моделей без нормализации, с MinMax и Standard масштабированием
- Подбор гиперпараметра `alpha` (0.001 → 100) на полиномиальных признаках (degree=10) для демонстрации переобучения
- Сравнение с наивными базовыми моделями (предсказание среднего и медианы)

### 4. Дополнительные техники
- Логарифмирование целевой переменной (`log1p`) для улучшения распределения
- Удаление выбросов из обучающей выборки по правилу IQR

## Результаты

Лучшие модели по метрикам на тестовой выборке:

| Метрика | Лучшая модель | Значение |
|---|---|---|
| MAE | [MinMax] Lasso (sklearn) | 713.32 |
| RMSE | [MinMax] Ridge (sklearn) | 1041.80 |
| R² | Ridge (sklearn) | 0.5714 |

Самая стабильная модель (минимальный разрыв train/test по R²): **ElasticNet (sklearn)** — gap 0.0067.

**Ridge + log(target):** MAE test = 702.84, R² test = 0.5652 — лучший MAE среди всех конфигураций.

**Ключевые наблюдения:**
- Кастомные SGD-реализации уступают sklearn при малом числе итераций, но сходятся к аналогичным результатам при достаточном обучении
- MinMax-скейлинг с ElasticNet даёт нестабильный результат из-за чувствительности к масштабу регуляризации
- Полиномиальные признаки (degree=10) без достаточной регуляризации приводят к катастрофическому переобучению (R² ≈ -10²³)
